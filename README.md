# CI/CD Integration

This a CI/CD learning experience with Bash scripts testing and a very basic REST API.


## How to run
In your terminal, run this: `docker pull christellaa/ci-cd-integration-img`

Then, run this: `docker run -d -p 8081:8081 christellaa/ci-cd-integration-img:latest`

You access the website by writing in the address bar: `http://localhost:8081/`
